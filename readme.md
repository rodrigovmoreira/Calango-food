
# Calango Food

## Sobre o Projeto

Calango Food é uma plataforma de delivery inovadora que integra o WhatsApp como canal principal para pedidos e rastreamento de entregas. O sistema oferece uma experiência simplificada e acessível para clientes e restaurantes.

## Características Principais

- **Integração WhatsApp**: Realização de pedidos diretamente via WhatsApp
- **Rastreamento em Tempo Real**: Acompanhamento da entrega pelo mensageiro
- **Interface Intuitiva**: Fácil navegação para clientes
- **Gestão de Pedidos**: Painel para restaurantes gerenciarem solicitações

## Tecnologias Utilizadas

- Node.js / Python
- WhatsApp API
- Banco de Dados
- Frontend Web

## Instalação

```bash
git clone https://github.com/seu-usuario/calango-food.git
cd calango-food
npm install
```

## Como Usar

1. Configure as credenciais da API WhatsApp
2. Inicie o servidor
3. Adicione o bot do Calango Food aos seus contatos
4. Envie uma mensagem para começar
